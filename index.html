Ploperty

What are the property trends in your neighbourhood?

There is vast, publicly available data on the Singapore real estate market, but it often hard to distill the raw data into user-friendly statistics. Ploperty is a website that scrapes live public data and distills it back to the viewer in a meaningful way. It provides data analytics and visualization for property rentals in Singapore based on neighborhood.

Built with Angular 2, Node.js, and Mapbox.

Startup SG

A Project for the Startup Community in Singapore

Startup SG is an API that helps visualize the relationships between co-working spaces, incubators, investors, events, and goverment programs. It gives viewers an overview of the local startup ecosystem and is relevant for anyone interested to learn more about startups, investing, or ongoing events. The information can be updated live and crowdsourced so that the site will grow organically as the startup scene in does.

Built with Node.js and uses Google Maps and Algolia APIs.

IOU

Owe your friend a beer for helping you with a favor? Buy them one here!

Owe your friends gifts? IOU allows you to document what you owe or gift your friends, and vice versa. It can be any item in any quantity, and is a playful way to keep track of what you’ve been actually (or virtually) giving out to friends!

Built with Ruby on Rails.
